# Arcade AI Worlds

## Project info

This project is a Vite + React + TypeScript app.

## How can I edit this code?

You can use any editor and local workflow.

### Use your preferred IDE

Clone the repository and work locally.

Requirements: Node.js and npm (or another compatible package manager).

```sh
# Step 1: Clone the repository
# Step 2: Go to the project directory
cd frontend/arcade-ai-worlds-main

# Step 3: Install dependencies
npm install

# Step 4: Start the dev server
npm run dev
```

## Build and preview

```sh
npm run build
npm run preview
```
